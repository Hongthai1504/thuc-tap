#### Sử dụng terminal trong dự án

- npm init -y
  - Tạo package.json để quản lý thông tin dự án
- npm install express mysql2 cors bcrypt jsonwebtoken
  - Cài đặt các thư viện
    - express: Giúp tạo Server nhanh chóng.
    - mysql2: Dùng để nói chuyện với Database XAMPP.
    - cors: Cho phép giao diện (Frontend) kết nối vào Server mà không bị chặn.
    - bcrypt & jsonwebtoken: Xử lý băm mật khẩu và đăng nhập theo đúng chuẩn bảo mật của tài liệu.
